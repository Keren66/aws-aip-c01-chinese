# AWS AIP-C01 生成式AI开发专家 - 中文互动刷题

AWS Certified Generative AI Developer - Professional (AIP-C01) 考试的中文互动版刷题工具。

## 内容

- **119 道真题**，覆盖全部 5 大类别
- **全部中文翻译**：题目、选项、解析均翻译为自然中文
- **交互式 HTML**：无需网络，浏览器直接打开即可使用

## 分类

| 类别 | 题数 |
|------|:----:|
| 基础模型集成、数据管理和合规 | 29 |
| 实施与集成 | 24 |
| 运营效率与 GenAI 优化 | 26 |
| 测试、验证与故障排除 | 12 |
| AI 安全、安全性与治理 | 26 |

## 功能

- 分类筛选过滤
- 关键词搜索（题目/选项/解析）
- 随机抽题
- 书签收藏（本地存储）
- 显示/隐藏答案及解析
- 暗色/亮色主题切换
- 键盘导航（← → 翻页，R 随机）
- 移动端适配

## 使用方式

直接打开 `index.html` 即可使用，所有数据内嵌在文件中，无需服务器。

## 数据来源

AWS AIP-C01 认证考试参考资料。
