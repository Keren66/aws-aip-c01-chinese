# AWS AIP-C01 生成式AI开发专家 - 中文互动刷题

AWS Certified Generative AI Developer - Professional (AIP-C01) 考试的中文互动版刷题工具。

## 🌐 在线使用（推荐）

直接访问 GitHub Pages 在线版，无需下载安装：

**https://keren66.github.io/aws-aip-c01-chinese/**

手机、电脑、平板都能用，打开即刷。

## 📥 下载使用

### 方式一：下载整个仓库

```bash
git clone https://github.com/Keren66/aws-aip-c01-chinese.git
cd aws-aip-c01-chinese
```

然后用浏览器打开 `index.html` 即可。

### 方式二：只下载 HTML 文件

1. 打开 https://raw.githubusercontent.com/Keren66/aws-aip-c01-chinese/main/index.html
2. 按 `Ctrl+S` 保存到本地
3. 用浏览器打开保存的文件

### 方式三：下载 ZIP 压缩包

1. 打开 https://github.com/Keren66/aws-aip-c01-chinese
2. 点击绿色按钮 **Code** → **Download ZIP**
3. 解压后打开 `index.html`

### 方式四：直接用 AI 部署

如果不知道怎么部署，直接把仓库链接发给 AI（Claude、ChatGPT 等），说：

> "帮我把这个项目部署一下"

AI 会自动帮你完成所有操作。现在连部署都可以交给 AI 做了。
